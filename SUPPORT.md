## Common Help Topics

### 1. Restore Premium Purchase

If you already paid but premium is not unlocked:
- Open Dayring.
- Go to Settings.
- Tap "Restore Purchases."
- Make sure you are signed into the same Apple ID used for the purchase.

### 2. Notifications Not Appearing

- Open iOS Settings > Notifications > Dayring.
- Ensure notifications are allowed.
- In Dayring, verify reminders are enabled for tasks.

### 3. Calendar Events Not Showing

- Open iOS Settings > Privacy & Security > Calendars > Dayring.
- Allow access.
- In Dayring Settings, turn on calendar integration.

### 4. Widget Not Updating

- Open Dayring once to refresh data.
- Remove and re-add the widget.
- Confirm the app is not restricted by Low Power Mode or Background limits.

### 5. Data Export / Import

In Dayring Settings:
- Use "Prepare Export..." to export your data as JSON.
- Use "Import Data (JSON)" to import previous exports.
